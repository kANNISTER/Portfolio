    Blood Oxygen Monitoring System

    A low-cost IoT-enabled web application was developed to monitor heart rate and blood oxygen levels. The server, built with Node.js, Express, and MongoDB, used RESTful APIs for secure data transmission and device management. A React-based front-end provided real-time data visualization with daily and weekly summaries. The MAX30102 sensor was integrated with a Photon development board, utilizing synchronous state machines for accurate data collection. Configurable measurement intervals allowed customized data capture, while offline storage enabled automatic synchronization upon Wi-Fi

    Designed and implemented a Random Forest Classifier that achieved 87% accuracy in predicting ODI match outcomes, player performances, and key events. Developed a real-time dashboard that simulated live match updates, dynamically integrating changing variables with predictions refreshed every five minutes. Built backend APIs to simulate live data using pre-existing datasets, ensuring accurate and timely match insights. This interactive system enhanced user engagement by providing continuously evolving predictions, offering cricket fans an immersive and data-driven match experience.

Built APIs for key match dynamics such as Batting Performance, Bowling Performance, and Partnerships, ensuring seamless live updates and in-depth analysis for end users.
